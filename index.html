How the Web Works

The world wide web is a web of intercommunicating computers. someone opens a website like www.google.com they are really sending a HTTP request to the website's server. This request tells the server to send a HTML file back in response. This file is then interpreted by your computer's web browser displaying a website. This video does a good job of explaining this.

HTML

HTML stands for Hyper Text Markup Language. HTML contains text which is rendered on your computers. HTML files make up most of the files on the intrenet. Text Content is "what you see" and Markup is "how it looks". An overview of this is available here.

Tags and Elements

Although you can type plain text and it will render as text in HTML their are many limitations to that method. A more effective way to do it is elements. Elements consist of 3 parts, an opening tag, content block, and a closing tag. This video explains this concept more in-depth..

Why Computers are Stupid

Computers are literal. What you tell them to do is what they do. This causes problems when you make a typo or mistake, because it can cause huge errors in a program.

Inline vs Block Elements

Inline elements are, as the same suggests in line. Block elements create an "invisible box" around content inside of them.
